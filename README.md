# Gerenciador_medicamentos_proj

Problema: Idosos esquecem horários ou tomam doses erradas.

Solução: CLI simples para agendamento e controle de estoque de remédios.

Versão: 1.0.0 (SemVer)

🚀 Como rodar
1- pip install -r requirements.txt
2- python src/app.py

🧪 Testes e Qualidade
1- Para rodar testes: pytest
2- Para rodar o linter: ruff check .
